# Before you can run these skills

1. **Figma access.** You need access to the DevRel Diagram System file in the
   Oracle DevRel team project. Open it and **duplicate it** into your own copy
   (File menu, Duplicate). You will work in your copy, never the shared base.
2. **Oracle Sans installed locally** so text renders correctly. The system uses
   `Semi Bold` wherever the spec says "Medium" (Oracle Sans has no Medium weight).
3. **Figma MCP connected** in your agent, and the **figma-use** engine skill
   installed (see README companion installs).

The first time you run `devrel-diagram`, it asks for your copy's URL and saves it
locally. After that it always targets your copy.
